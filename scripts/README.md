# Scripts!

## [restart_all.sh](restart_all.sh)

'restart_all.sh' essentially goes through every single docker container, kills it, and then starts it up again. It also displays the status of the Fail2Ban daemon.

To Execute:
```
./
```

## [down_all.sh](down_all.sh)

'down_all.sh' goes through through every single docker container and kills it.

To Execute:
```
./
```
